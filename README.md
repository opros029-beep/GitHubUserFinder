## **Описание программы**

*"GitHub User Finder — это графическое приложение для поиска пользователей GitHub.
 Позволяет находить информацию о пользователях по логину, добавлять их в избранное, сохранять список избранных в JSON-файл и быстро открывать профили в браузере.
 Приложение решает проблему быстрого поиска и сохранения информации о разработчиках на GitHub."*

---

## **Требования для запуска**

Что нужно установить на компьютер:
- **Python** 3.8 или выше
- **Библиотеки:**`pip install requests`

## **Как запустить**
###**Способ 1 (если есть git):**
```bash 
git clone https://github.com/ваш-логин/github-user-finder.git
cd github-user-finder
python github_user_finder.py
```
###**Способ 2 (без git):**
1.Скачайте файл github_user_finder.py

2.Откройте терминал в папке с файлом

3.Выполните команду:
```
bash
python github_user_finder.py
```
##**Как пользоваться**

###**Пошаговая инструкция:**
1.**Поиск пользователя:**

 -Введите логин пользователя GitHub в поле поиска

-Нажмите кнопку «Найти» или клавишу Enter

-Результат отобразится на вкладке «Результаты поиска»

2.**Просмотр информации:**

-В таблице отображаются: логин, имя, количество подписчиков и репозиториев

-Дважды щелкните правой кнопкой мыши для вызова контекстного меню

3.**Добавление в избранное:**

-На вкладке «Результаты поиска» щелкните правой кнопкой мыши по пользователю

-Выберите «Добавить в избранное»

-Пользователь сохранится на вкладке «Избранное»

4.**Удаление из избранного:**

-Перейдите на вкладку «Избранное»

-Щелкните правой кнопкой мыши по пользователю

-Выберите «Удалить из избранного»

5.**Открытие профиля в браузере:**

-Щелкните правой кнопкой мыши по пользователю (в любой вкладке)

-Выберите «Показать профиль»

-Откроется страница пользователя на GitHub.com

##**Тестирование**

| Ошибка | Причина | Решение |
|--------|---------|---------|
| «Поле поиска не должно быть пустым» | Не введен логин | Введите имя пользователя |
| «Пользователь не найден» | Неверный логин | Проверьте правильность написания |
| «Нет подключения к интернету» | Отсутствует интернет | Проверьте соединение |
